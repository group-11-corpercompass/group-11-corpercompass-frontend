# corpercompass-frontend
A digital travel, transition and experience support platform for NYSC members in Nigeria.

Frontend project built with:
- HTML
- CSS
- JavaScript

## Folder Structure

corpercompass/ -> Website pages
CSS/ -> Stylesheets
JS/ -> JavaScript files
assests_images/ -> Images, fonts, icons
Components/ -> Reusable UI sections
